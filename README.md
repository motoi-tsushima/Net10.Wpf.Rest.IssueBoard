# README

このプログラムは、.NETアーキテクチャの種類を解説するために、作成したサンプルプログラムです。  
その、.NETアーキテクチャの種類を解説している記事が以下になります。全サンプルプログラムの一覧も掲載しています。

[.NET 10 の画面有りアーキテクチャの解説とサンプルコード](https://snow-stack.net/dotnet10-GUI-architecture-list/)

記事に記載する全てのサンプルプログラムは、課題管理機能を簡易的に実現しています。  
.NETアーキテクチャのサンプルを簡単に説明するために、必要最小限の機能しか実装していません。  
よって、実用的なプログラムとは言えません。  
アーキテクチャの説明が目的なので、閲覧者がソースコードを読みやすいようにしています。  
よって、配布やインストール・リリース機能などは、作成していません。  
単純にするため、ユーザー認証など一般的セキュリティ要件を満たす機能は最初から実装していません。

その点をご了承ください。

## Database & Table

サンプルプログラムは、ローカルPCに SQL Server のデータベースとテーブルが存在する事を前提に作成しています。  
参照するデータベースとテーブルのDDLは、以下のリポジトリで公開しています。

[DATABASE.Samples.Issues](https://github.com/motoi-tsushima/DATABASE.Samples.Issues)
